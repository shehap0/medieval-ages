# Citadel of Vael Anore

A single-page marketing landing page for a fictional medieval archive and strategic-records society. Built with Astro, Three.js, and GSAP.

## Tech Stack

- **Astro** — static site generation
- **Three.js** — 3D artifact (brass astrolabe)
- **GSAP** — scroll-triggered animations

## Commands

| Command | Action |
|---------|--------|
| `npm run dev` | Start dev server |
| `npm run build` | Build to `dist/` |
| `npm run preview` | Preview production build |

## Design

Heraldic-archive aesthetic. Deep cobalt-indigo palette with bronze/brass accents. Typography: Marcellus (display), Spectral (body), JetBrains Mono (ledger). See `PRODUCT.md` for full design rationale.
